# 💸 Expense Splitter (Splitwise Clone)

A modern and premium expense splitting web app where users can manage group expenses, track balances, and settle payments easily.

---

## ✨ Features

- 🔐 Authentication (Login / Signup)
- 👥 Add members (no account required)
- 💰 Add & split expenses
- ⚖️ Equal / custom split support
- 📊 Settlement summary (who owes whom)
- ✅ Record payment (Settle Up feature)
- 🎨 Premium modern UI (glassmorphism + dark theme)
- 📱 Responsive design

---

## 🧠 How It Works

- Add group members by name  
- Add expenses (who paid & who participated)  
- App calculates balances automatically  
- Shows:
  - Who owes whom  
  - How much to settle  

---

## 🛠️ Tech Stack

- Frontend: React / Vite
- Backend: Node.js + Express
- Database: MongoDB
- Styling: Tailwind CSS
