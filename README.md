# 💸 Payment Splitter (Splitwise Clone)

A modern full-stack expense splitting application that helps users manage group expenses, track balances, and settle payments easily.

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 👥 Add members without account (name-based)
- 💰 Add & split expenses
- ⚖️ Equal / custom split support
- 📊 Settlement summary (who owes whom)
- 💳 Record payment (Settle Up feature)
- 🎨 Premium UI (dark theme + glassmorphism)
- 📱 Fully responsive design

---

## 🧠 How It Works

1. Create a group  
2. Add members (no login required for others)  
3. Add expenses  
4. App automatically calculates balances  
5. See who owes whom  
6. Record payments and settle up  

---

## 🛠️ Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MongoDB
- Styling: Tailwind CSS
